**Goal**
TMD-26631 Verify Fluid Flow Rate field properties in Cooling Unit.
---
## Test Steps
1. Go To Items List
2. **Verify Fluid Flow Rate Column Is Hidden By Default**
3. Open Show/Hide Column
4. Select Fluid Flow Rate
5. Click Apply Button
6. **Verify Fluid Flow Rate Column Is Visibile In Items List**
7. Go To Models Library
8. **Verify Fluid Flow Rate Column Is Hidden By Default**
9. Click Show/Hide Column
10. Select Fluid Flow Rate
11. Click Apply Button
12. **Verify Fluid Flow Rate Column Is Visibile In Models Library**
13. Go To Field Management Page
14. Click Standard Fields Subtab
15. Filter Fluid Flow Rate On Label
16. **Verify Fluid Flow Rate Is Visible**
